## Que revisar

- Titulo claro: producto, modelo, medida, beneficio.
- Fotos: primera foto clara, detalles, medidas o escala.
- Descripcion: que incluye, para quien sirve, compatibilidad y limites.
- Precio: costo real, comision, envio, impuestos y margen.
- Respuestas: stock, envio, medidas, garantia, factura, precio, cambios y reclamos.
- Postventa: datos minimos para resolver problemas.

## Senal de mejora

Si bajan las preguntas repetidas y suben las consultas concretas, la publicacion esta trabajando mejor.

Si siguen preguntando lo mismo, ese dato debe estar mas arriba en la publicacion.
